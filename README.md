# SCP-imputation-batchcorrection-benchmark
Bioinformatic pipeline for single-cell proteomics data analysis: benchmarking of missing value imputation strategies (KNN, QRILC, limpa) and batch effect correction methods (removeBatchEffect, ComBat) on the Leduc et al. (2022) pSCoPE dataset. Implemented in R using the scp/QFeatures Bioconductor ecosystem.
